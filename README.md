# Fantasy Basketball Tracker
The app leverages the ESPN Fantasy Basketball API to help you build and manage your fantasy team throughout the season. Get data-driven insights and strategic pointers to optimize your roster decisions!


## Features

- **ESPN API Integration**: Fetches real-time fantasy basketball data from ESPN
- **Team Analysis**: Track your team's performance and stats
- **Season-Long Insights**: Get helpful pointers and recommendations throughout the fantasy season
